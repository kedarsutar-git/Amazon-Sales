# Amazon Sales Data Analysis Project

## Overview
This project focuses on analyzing an Amazon Sales dataset using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn. The main goal of the project is to clean, explore, visualize, and understand sales data to discover useful business insights and customer purchasing patterns.

The dataset contains product information such as product name, category, actual price, discounted price, discount percentage, ratings, reviews, and product details. The project includes data cleaning, handling missing values, descriptive statistics, and data visualization techniques.

---

## Features
- Data Cleaning using Pandas
- Exploratory Data Analysis (EDA)
- Statistical Analysis of product prices and ratings
- Visualization using charts and graphs
- Understanding customer purchasing behavior
- Finding relationships between discount, price, and ratings

---

## Key Insights
- Products with lower prices generally received higher ratings.
- Most customers purchased products in the lower discounted price range.
- Higher actual price products often had larger discounts.
- Discounted products still maintained good customer ratings.
